ESP32 Maze-Solving Robot

An autonomous robot that explores and solves a physical maze in real time. Ultrasonic sensors handle wall detection, an L298N H-bridge driver controls the drive motors, and a breadth-first search (BFS) algorithm running on the ESP32 computes the shortest path once enough of the maze has been mapped.

A WebSocket server on the ESP32 streams the robot's live position and discovered maze layout to a browser-based visualizer, so the maze-solving process can be watched in real time from any device on the network.

  • ESP32
  • Ultrasonic Sensors
